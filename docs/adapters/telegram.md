# Telegram Adapter

## Connection

Long polling with optional HTTP proxy support.

## Setup

1. Talk to [@BotFather](https://t.me/BotFather) on Telegram
2. Create a new bot with `/newbot`
3. Get the **Bot Token**
4. (Optional) Configure HTTP proxy in config

## Capabilities

| Feature | Supported |
|---------|-----------|
| Text | ✅ |
| File send | ✅ |
| Image send | ✅ |
| Audio/Voice | ✅ |
